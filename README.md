# 🌤️ Real-Time Weather App

Ek modern Weather Application jo OpenWeatherMap API ka istemal karte hue kisi bhi shehar ka mausam batati hai. Ismein user ki auto-location ka feature bhi shamil hai.

## ✨ Key Features
- **Auto-Location Detection:** Page load hote hi user ka shehar khud detect karta hai.
- **Search by City:** Dunya ke kisi bhi shehar ka mausam search karein.
- **Dynamic Backgrounds:** Mausam ke hisaab se (Clear, Rain, Clouds) background color change hota hai.
- **Responsive UI:** Glassmorphism design jo mobile aur desktop dono par fit aata hai.

## 🛠️ Tech Used
- HTML5 & CSS3 (Glassmorphism & Flexbox)
- JavaScript (Async/Await & Geolocation API)
- OpenWeatherMap API
